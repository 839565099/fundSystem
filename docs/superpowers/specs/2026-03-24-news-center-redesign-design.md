# 资讯中心重新设计

## 概述

重新设计资讯中心，实现新闻智能分类、完善详情内容、突出来源展示。

## 需求

1. **按主题智能分类** - 根据新闻标题/内容自动识别分类
2. **详情内容完善** - 能抓取正文显示全文，否则显示摘要并突出原文链接
3. **来源展示** - 显示来源名称+图标
4. **布局** - 顶部Tab切换分类

## 设计方案

采用后端智能分类方案，爬虫保存时自动打分类标签。

### 1. 后端改造

#### 1.1 分类规则

`NewsCrawlerService` 新增 `classifyNews()` 方法：

| 关键词 | 分类 |
|--------|------|
| 基金、净值、申购、赎回、基金经理 | 基金动态 |
| 政策、监管、证监会、央行、银保监 | 政策法规 |
| 市场、行情、大盘、指数、A股 | 市场分析 |
| 行业、板块、概念、龙头 | 行业动态 |
| 资金、流入、流出、主力、北向 | 资金流向 |
| 其他（默认） | 财经要闻 |

#### 1.2 全文抓取增强

`NewsCrawlerService` 新增 `fetchFullContent()` 方法：
- 根据原文链接尝试抓取正文
- 使用 Jsoup 解析网页，提取正文内容
- 设置超时和重试机制
- 失败时保留摘要作为内容

#### 1.3 爬虫流程调整

```
1. 调用外部API获取新闻列表
2. 对每条新闻：
   - 调用 classifyNews() 智能分类
   - 调用 fetchFullContent() 尝试获取全文
   - 保存到数据库
```

### 2. 前端改造

#### 2.1 News.vue - 资讯列表页

**分类Tab**：
- 固定分类：全部、财经要闻、基金动态、市场分析、政策法规、行业动态、资金流向
- 点击切换时请求后端筛选

**新闻卡片**：
- 增加来源图标（新浪财经/东方财富）
- 来源名称使用主题色突出显示

#### 2.2 NewsDetail.vue - 详情页

**来源区域**：
- 增加来源图标
- 来源名称+图标组合展示

**内容区域**：
- 有全文时正常显示
- 内容不足时（<100字）显示提示+"查看原文"按钮

**原文按钮**：
- 更醒目的样式
- 始终显示（如果有链接）

### 3. 来源图标

| 来源 | 处理方式 |
|------|----------|
| 新浪财经 | 使用 `https://www.sina.com.cn/favicon.ico` 或内嵌SVG |
| 东方财富 | 使用 `https://www.eastmoney.com/favicon.ico` 或内嵌SVG |

考虑网络加载问题，优先使用内嵌SVG图标或 Naive UI 的图标。

## 验收标准

1. 新闻能按主题自动分类，准确率 > 80%
2. 分类Tab点击可正常筛选
3. 来源图标正确显示（新浪/东财）
4. 详情页有原文链接时显示醒目的"查看原文"按钮
5. 内容不足时有明确提示
